# Storage Patent Deepdive

对一批存储领域专利进行系统分析，涵盖分类标引、价值评分、Claim 1 要件分解、竞争格局分析、技术趋势研判，输出 Word 报告和配套 Excel 底表。

---

## 是什么

上传一份专利清单，AI 自动完成全套分析流程：补全数据 → 分类标引 → 每件专利技术方案一句话摘要 → 价值评分排序 → 重点专利 Claim 1 要件级分解 → 竞争格局深度分析（含技术路线对比矩阵 + 申请人画像）→ 技术趋势研判 → 布局建议，输出 Word 报告和 Excel 底表。

适用领域：存储（SSD、HDD、磁带、NVMe、NAND、分布式存储、文件系统等）。

## 使用方式

提供专利清单后，触发示例：

- "帮我分析这批专利"
- "做专利竞争格局分析"
- "patent deepdive"
- "分析一下友商的专利布局"

输入要求：专利清单 Excel，至少包含专利号。有标题、摘要、申请人、权利要求时分析更充分。可选参数为技术细分方向（如"磁带伺服"或"SSD 主控"），不指定则由 AI 自动判断。

---

## 输出说明

输出两个文件：

**1. Word 报告**（15-20 页），包含：
- 执行摘要（批次概况 + 核心发现）
- 技术分类全景（分类总览 + 方案分布概览）
- 竞争格局深度分析（申请人布局 + 技术路线对比矩阵 + 申请人画像 + 竞争空白）
- 重点专利深度剖析（前 30% 核心专利）
- 技术趋势与布局建议（P0/P1/P2）

**2. Excel 底表**，包含：
- 全量专利一览表（每件专利：专利号、申请人、分类、一句话技术方案、Claim 1 要件 JSON、创新类型、综合评分、价值等级）
- 分类分布统计
- 申请人分布统计

## 注意事项

- 全量专利的逐件要件分解输出到 Excel，Word 报告只聚焦整体性分析和重点专利深度剖析，二者互补不重复
- 分类标准由 AI 浏览全部专利后动态制定并提交用户审批，非固定模板
- 重点专利选取前 30%（按综合评分降序，至少 5 件）
- 趋势分析采用直白三段式：最新技术是什么 → 与老技术有何不同 → 业界向哪些方向发展，不使用抽象框架
