# Patent Idea Analyzer

专利Idea胶片质量预审与创新性分析工具。

---

## 是什么

上传一份专利Idea胶片（PPT或Word格式），自动完成两大块分析，输出预审报告：

1. **胶片撰写质量检查**：按6项标准逐一审核，找出撰写问题，给出改进意见
2. **创新性深度分析**：三步法提炼发明点，检索3-5篇对比文件，评估创造性，给出保护点重写建议和实施例扩展方向

适用技术领域：存储（SSD、HDD、分布式存储、存储协议等）、AI（推理优化、AI存储、模型训练等），以及其他技术领域。

## 使用方式

上传胶片文件后，触发示例：

- "帮我分析这个专利胶片"
- "对这份Idea做质量预审"
- "这个Idea有没有创新性，帮我检索一下"

支持格式：`.pptx`、`.docx`

---

## 输出说明

输出分两步：

1. **Markdown 报告**（必出），文件命名 `{发明核心内容}-预审报告.md`，包含8章：
   - 封面信息
   - 发明点提炼（三步法结果）
   - 胶片撰写质量评价（6项检查表格）
   - 现有技术对比分析（3-5篇对比文件）
   - 创造性综合评估（★评分+风险点）
   - 保护点重写建议（诊断+示范）
   - 实施例扩展方向（发散/从属/延伸）
   - 总结与优先改进事项

2. **Word 报告**（按需转换），询问用户是否需要后生成 `.docx` 版本。

## 注意事项

- 创新性分析在质量检查之前执行，因为发明点提炼结果会影响保护点的评判
- 检索上限：5-10次搜索 + fetch 2-3篇全文，找到3-5篇有实质内容的对比文件后停止
- 保护点重写建议无论原有写法好坏都必须给出
- 若胶片内容不足5页或关键信息缺失，报告中需明确指出
