# Patent Classifier

按技术标签体系对一批专利进行分类标引，输出结构化 Excel 报告。

---

## 是什么

上传一份专利清单（Excel/CSV），指定标签体系（可使用内置的存储领域分类或自定义），AI 自动逐件分析专利内容，匹配最合适的分类标签，输出带置信度和分类理由的 Excel 报告。

支持领域：存储（SSD、HDD、NAND、分布式存储、存储协议等），以及其他技术领域。

## 使用方式

提供专利清单后，触发示例：

- "帮我给这批专利分类"
- "按存储技术标签体系标引这些专利"
- "对这些专利做技术分类"

支持格式：`.xlsx`、`.csv`

标签体系可选：内置存储领域 8 个一级分类，或由用户自定义提供。

---

## 输出说明

输出一份 Excel 文件，包含以下列：

| 列名 | 说明 |
|------|------|
| 专利号 | 专利号 |
| 标题 | 标题 |
| 申请人 | 申请人 |
| 一级分类 | 最佳匹配的一级类别 |
| 二级分类 | 最佳匹配的二级类别 |
| 置信度 | 高/中/低 |
| 理由 | 分类依据（1-3句） |
| 分类来源 | 自动分类 / 用户已有 / 同族推断 |

同时对话中输出分类分布概览和质量概览。

## 注意事项

- 分类依据优先级：权利要求/技术方案 > 技术问题/功效 > 摘要 > 标题
- 匹配创新核心，不匹配应用场景：存储控制算法归硬件/工程，不归 AI 存储（即使服务于 AI）
- 专利数 ≥10 件时自动做同族聚类推断，同族专利分类应保持一致
- 边界 Case 可参考 SKILL.md 中的 7 个消歧示例
